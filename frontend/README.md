## Run Front End Locally

Clone the project

```bash
  git clone git@github.com:GuilhermeEsdras/projeto-sds.git
```

Go to the project frontend directory

```bash
  cd projeto-sds
  cd frontend
```

Install dependencies

```bash
  npm install
  //or
  yarn install
```

Start the server

```bash
  npm run start
  //or
  yarn start
```
